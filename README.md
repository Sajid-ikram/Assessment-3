

# Airline Ticket Booking System

This project is a prototype for a low-cost airline's booking system, developed for both web and mobile platforms using Flutter, Firebase, Node.js, and MongoDB. The key feature is an optimized seating plan algorithm that prevents single scattered seats, ensuring efficient seat allocation for groups and individuals.

## Features

- **Seating Algorithm:** Minimizes single scattered seats to enhance group bookings.
- **UML Documentation:** Comprehensive use case, activity, sequence, and class diagrams.
- **TDD/BDD:** Implemented Test-Driven Development and Behavior-Driven Development for robust testing.

## Technologies Used

- **Frontend:** Flutter
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Cloud Services:** Firebase

This project was the final project of the Advanced Topics in Software Engineering Module, demonstrating a plan-driven approach to software development, focusing on clear requirements and system modeling before implementation.
